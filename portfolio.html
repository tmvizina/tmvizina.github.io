Library Terminal

The Library Terminal was my first collaborative coding project in which two teammates and I worked together to develop a console app in C# that would display/check out books from a books list that would save state between runs of the program. I worked on the FileIO as well as writing out the LinQ query searches for selecting books from the list. I focused on the functionality of the program and learning to use to github coollabritively. It was a great experience to have to explain concepts to others that I just figured out and to try and prove to myself that I knew what I was talking about.

Block Buster Lab

The Blockbuster lab's purpose was to demonstrate knowledge of Composition as opposed to using inheritance for methods. It uses an interface and method stubs to allow for different children of the same parent class to call different methods without all the extra steps involved with standard inheritance

Pig Latin Translator

The Piglatin Translator was an early lab we did that asks for user input and returns the phrase entered translated into pig latin. It was great practice with string manipulation, but it is a very early project and it is almost all done entirely within the main method. Its not very condensed code and my understanding has grown a lot since this point
